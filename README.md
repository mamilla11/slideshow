# slideshow

### Create virtual environment:

#### Windows

```bash
python -m venv venv
```
```bash
venv/Scripts/activate
```

#### Linux

```bash
virtualenv -p /usr/bin/python3.9 venv
```
```bash
. ./venv/bin/activate
```

### Install dependencies:

```bash
pip install -r requirements.txt
```

### Run application:

```bash
python app/main.py
```
